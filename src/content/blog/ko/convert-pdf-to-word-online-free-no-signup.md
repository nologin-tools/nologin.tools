누군가 PDF를 보내왔어요. 수정해야 할 계약서일 수도 있고, 회사가 오래된 시스템에서 내보낸 가격표일 수도 있어요. 텍스트를 선택해서 Word에 붙여넣기 해보면 결과는 엉망진창이에요. 한 줄에 단어 하나씩, 공백은 사라지고, 표는 줄줄이 흩어진 셀로 폭발합니다.

이유가 있어요. 그리고 계정 없이도 제대로 처리할 수 있는 무료 도구가 있습니다.

## PDF를 Word로 변환하기 어려운 이유

PDF는 Word와 완전히 다른 방식으로 텍스트를 저장해요. Word 문서에는 단락, 스타일, 문서 구조가 있어요. "이건 제목이고", "이 셀들은 표에 속한다"는 정보가 담겨 있죠. PDF는 인쇄물에 가까워요. 페이지의 모든 문자 위치, 폰트, 글자 자체를 개별적으로 기록해요. 문장, 단락, 표 행이라는 개념 자체가 없어요. 문자들이 그냥 특정 좌표에 배치되어 있을 뿐이에요.

변환 도구가 PDF를 Word로 되돌릴 때는 그 구조를 역으로 추론해야 해요. 문자 위치를 보며 판단하죠. 이 문자들은 같은 줄에 있나? 이 줄들은 단락을 이루나? 이 숫자 열들은 표를 구성하나? Word나 InDesign으로 만든 깔끔한 텍스트 PDF라면 이 추론이 어느 정도 잘 돼요. 스캔 문서(본질적으로 텍스트 이미지)는 OCR이 필요해요. 사이드바, 각주, 이미지 주변에 텍스트가 배치된 복잡한 레이아웃의 PDF라면 출력 결과는 항상 불완전할 수밖에 없어요.

이건 특정 도구의 결함이 아니에요. PDF 형식 자체의 구조적 특성이에요. 미리 알아두면 많은 좌절을 피할 수 있어요.

## PDF24: 이메일도 묻지 않는 무료 온라인 PDF 변환기

아무것도 설치하지 않고, 계정도 만들지 않고 PDF를 Word로 변환하고 싶다면 [PDF24 Tools](/tool/tools-pdf24-org-en)부터 시작하세요. 완전 무료에 워터마크 없고, 일일 제한도 없고, 회원가입도 필요 없어요. 운영사는 독일의 geek software GmbH로 GDPR이 적용되며, 개인정보처리방침에 파일이 처리 후 서버에서 삭제된다고 명시되어 있어요.

PDF→Word 변환은 간단한 것부터 중간 복잡도의 문서까지 잘 처리해요. 텍스트가 단락으로 올바르게 흘러들어가고, 제목이 인식되고, 기본 표는 대부분 그대로 유지돼요. 출력은 Word, Google Docs, LibreOffice에서 열 수 있는 `.docx` 파일이에요. 깔끔한 비즈니스 레터나 텍스트 위주의 보고서라면 최소한의 수정만으로 바로 사용할 수 있어요.

PDF24는 대부분의 사람들이 언젠가 한 번쯤 필요로 하는 덜 일반적인 변환도 지원해요.

**PDF → Excel**: 원본 PDF에 재무 표나 데이터 그리드가 있으면 PDF24가 행과 열로 추출해요. 결과는 표가 어떻게 만들어졌느냐에 크게 달라져요. 원본 문서에서 실제 표 구조로 만들어진 건 깔끔하게 변환되지만, 레이아웃 소프트웨어에서 선으로 그린 표는 망가지는 경향이 있어요. 그래도 첫 시도로는 숫자를 직접 다시 입력하는 것보다 훨씬 낫죠.

**PDF → PowerPoint**: 누군가 프레젠테이션을 납작한 PDF로 공유했고 슬라이드를 편집해야 할 때 유용해요. PDF24는 각 페이지가 슬라이드 이미지가 되는 `.pptx`를 만들어요. 텍스트를 완전히 편집할 수는 없지만 순서 변경, 크기 조정, 주석 추가는 가능해요.

**PDF → JPG (및 기타 이미지)**: 각 페이지가 별도의 이미지 파일이 돼요. 특정 페이지를 다른 문서에 붙여넣거나, 전체 파일 없이 단일 페이지 미리보기를 보낼 때 유용해요.

이 모든 게 회원가입 없이 가능해요. 파일을 드롭하고, 변환 유형을 선택하고, 결과를 다운로드하면 돼요.

## Convertio: 엣지 케이스와 특이한 형식 조합에

PDF24 출력이 완전하지 않을 때 [Convertio](/tool/convertio-co)를 두 번째 선택지로 써볼 만해요. PDF→Word뿐 아니라 PDF→RTF, PDF→ODT, PDF→HTML, 그리고 PNG, TIFF, BMP, WebP 같은 이미지 형식까지 300개 이상의 형식 조합을 지원해요. 변환 엔진마다 특정 문서를 더 잘 처리하는 경우가 있어서, 같은 파일이어도 도구마다 눈에 띄게 다른 결과가 나올 수 있어요.

Convertio 무료 티어는 계정 없이 사용할 수 있어요. 파일 최대 100MB, 하루 두 번 변환 제한이에요. 가끔 쓰는 용도라면 충분해요. 더 많은 변환이 필요하다면 계정을 만들어 상한을 높일 수 있지만, 기본 무료 티어는 아무것도 필요 없어요.

특히 이미지→PDF나 PDF→이미지 변환에서 Convertio는 안정적이고 빠른 편이에요. 스캔한 페이지 여러 장을 단일 PDF로 합치거나, 여러 페이지 PDF를 고해상도 PNG 시퀀스로 내보낼 때 깔끔하게 처리해줘요.

## TinyWow: 파일 문제가 하나 이상일 때

[TinyWow](/tool/tinywow-com)는 순수한 PDF 변환기보다 더 넓어요. PDF, 이미지, 동영상, 텍스트 파일을 다루는 회원가입 없는 툴킷이에요. "이 PDF를 Word로 변환하고, 거기 있는 로고 이미지 크기도 바꾸고, 누가 보낸 동영상도 MP4로 변환해야 해"라는 상황이라면 TinyWow가 탭 전환 없이 세 가지 모두 처리해줘요.

PDF 변환만 놓고 보면 PDF24와 같은 기본 기능을 지원해요. Word, Excel, JPG, PNG로 변환 가능해요. 광고가 있어서 PDF24보다 덜 깔끔하지만 실제 도구를 막지는 않아요. 워터마크 없고, 계정 불필요해요.

TinyWow의 PDF 편집 도구도 언급할 가치가 있어요. 페이지 어디에나 텍스트 박스, 도형, 이미지를 배치할 수 있어요. 이건 변환이 아닌 추가 편집이에요. 기존 PDF 위에 내용을 겹쳐 올리는 방식이에요. 스캔된 양식(폼 필드가 아닌 스캔본)을 채우거나 도장을 찍거나 서명 이미지를 삽입할 때 유용해요. [브라우저에서 PDF 편집하기 가이드](/blog/edit-pdf-without-installing-anything)에서 같은 방식을 자세히 다루고 있어요.

## 스캔 PDF 처리: OCR 문제

스캔 PDF가 가장 어려운 케이스예요. 이건 텍스트가 아니라 종이의 이미지예요. PDF 변환기는 기계 가독 문자로 존재하지 않는 것은 추출할 수 없어요. 일반 PDF 변환기로 스캔된 임대계약서를 Word로 변환하면 깨진 텍스트가 나오거나, 페이지 이미지만 담긴 Word 파일이 만들어져요.

해결책은 변환 전에 OCR(광학 문자 인식)을 먼저 실행하는 거예요. PDF24에는 OCR 도구가 내장되어 있어요. 스캔 PDF를 업로드하고 OCR을 실행하면 문자가 기계 가독 상태가 된 텍스트 레이어 PDF가 만들어져요. 그 다음에 PDF→Word 변환을 하면 제대로 작동해요.

OCR 품질은 스캔 품질에 달려 있어요. 인쇄 문서의 깔끔하고 고해상도 스캔은 잘 변환돼요. 불량한 조명에서 찍은 손글씨 양식 사진은 그렇지 않을 수 있어요. PDF24의 OCR은 100개 이상의 언어를 지원하며 역시 무료에 계정 불필요해요.

[iLovePDF](https://www.ilovepdf.com/)는 OCR 분야에서 특별히 언급할 만한 또 다른 선택지예요. 스캔 PDF를 업로드하면 Word 문서를 바로 받을 수 있는 OCR→Word 변환을 단일 단계로 제공해요. 기본 사용은 로그인이 필요 없고 OCR 모드에서 15개 언어를 지원해요.

## 용도별 도구 선택

| 작업 | 첫 번째 선택 | 참고 |
|------|-----------|------|
| PDF → Word (텍스트 문서) | PDF24 | 제한 없음, 계정 불필요 |
| PDF → Excel (표) | PDF24 또는 Convertio | 출력 정리가 필요하면 둘 다 시도 |
| PDF → JPG/PNG | PDF24 또는 Convertio | 변환 전 해상도 선택 |
| 스캔 PDF → Word | PDF24 (OCR 먼저) | OCR 단계 필수 |
| PDF → PowerPoint | PDF24 | 페이지가 슬라이드 이미지로 변환 |
| Word/Excel/PPT → PDF | PDF24 또는 TinyWow | 이 방향도 둘 다 신뢰성 높음 |
| 다양한 형식 한꺼번에 | TinyWow | 더 넓은 툴킷, 광고 있음 |

## 반대 방향: 아무것도 설치하지 않고 PDF 만들기

같은 도구들이 반대 방향 변환도 지원해요. 이쪽이 보통 더 쉬운 경우예요. Word→PDF, Excel→PDF, PowerPoint→PDF — 이 변환들은 무손실이에요. PDF는 편집 가능한 형식이 아닌 최종 출력 형식이라서, 원본 문서의 구조가 고정 레이아웃 PDF 형식으로 깔끔하게 매핑돼요.

PDF24는 Word, Excel, PowerPoint, 이미지 파일을 정확하게 PDF로 변환하며 계정이 필요 없어요. 이미 Google 생태계에서 작업하고 있다면 Google Docs, Google Sheets, Google Slides에서도 파일→다운로드→PDF로 직접 내보낼 수 있어요. 추가 도구가 필요 없어요.

## 서버에 남는 것들 (알아둬야 할 점)

여기서 소개하는 모든 변환은 처리를 위해 파일을 원격 서버에 업로드해요. 이건 무료 도구만의 특성이 아니에요. 유료든 무료든 대부분의 온라인 변환기가 마찬가지예요. 중요한 건 결과를 다운로드한 후 파일이 얼마나 오래 서버에 남느냐예요.

PDF24는 개인정보처리방침에 따라 처리 후 서버에서 파일을 삭제해요. Convertio는 변환 후 1시간이면 파일이 삭제된다고 명시해요. TinyWow는 30분 후 삭제해요. 이 정도 보관 기간은 합리적이지만 0은 아니에요.

기밀 문서 — 고용계약서, 의료 기록, 재무 데이터 — 에 대해서는 모든 온라인 도구를 신뢰할 수 없는 제3자로 취급해야 해요. 정말로 오프라인 처리가 필요하다면 PDF24는 무료 Windows 데스크톱 앱(PDF24 Creator)을 제공하는데 변환이 완전히 로컬에서 실행돼요. [Sejda](https://www.sejda.com/)도 서버가 아닌 내 컴퓨터에서 파일을 처리하는 데스크톱 버전을 제공해요.

그 외 다른 경우들 — 수정 중인 견적서, 숫자를 뽑아야 하는 일정표, 작성해야 할 양식 — 에서는 로그인 없는 온라인 도구로 충분해요. 파일 유형에 맞는 도구를 선택하면 대부분의 변환이 1분도 안 걸려요.

쓸 수 있는 도구가 많아질수록 막히는 경우가 줄어들어요. 대부분의 작업엔 PDF24, 다른 변환 엔진이나 특이한 형식이 필요할 때는 Convertio, 뒤섞인 파일 문제가 있을 때는 TinyWow. 어느 것도 이메일 주소를 요구하지 않아요. 모두 일을 해결해줘요.

PDF는 1993년에 설계된 포맷이에요. 어떤 컴퓨터에서 열어도 동일한 폰트, 동일한 레이아웃으로 보이게 하자는 목표로 만들어졌죠. 최종 문서를 공유하기에는 완벽했지만, 편집은 완전히 다른 이야기였어요. 30년 동안, PDF를 수정하려면 Adobe Acrobat(월 24.99달러)을 구독하거나, 크랙 버전을 쓰거나, 처음부터 Word 문서를 새로 만들어야 했어요.

이제는 달라졌어요. 완전히는 아니지만 — 진지한 편집은 여전히 데스크탑 소프트웨어가 더 강력합니다 — 하지만 대부분의 사람들이 실제로 필요로 하는 작업(파일 병합, 큰 PDF 압축, 양식 작성, 주석 추가)은 브라우저 탭 하나에서 거의 다 처리할 수 있어요. 계정 없이도요.

작업에 맞는 도구를 선택하는 방법을 알아볼게요.

## 정확히 무엇을 편집해야 하는지 먼저 파악하세요

「PDF 편집」은 사실 아주 다양한 작업을 포함해요. 올바른 도구를 선택하려면 실제로 무엇이 필요한지 알아야 해요.

**구조 편집** — 여러 PDF를 하나로 병합하기, 큰 PDF 분할, 페이지 순서 변경, 페이지 회전 — 은 온라인에서 가장 쉬운 작업이에요. 모든 도구가 잘 처리하고, 결과도 안정적입니다.

**압축과 변환** — 이메일 전송을 위해 큰 PDF 줄이기, Word/Excel/JPG로 변환 — 도 폭넓게 지원돼요. 원본 문서의 복잡도에 따라 품질이 달라지지만, 일반적인 용도라면 온라인 도구로 충분합니다.

**양식 작성과 주석** — 폼 필드에 텍스트 입력, 텍스트 하이라이트, 서명 이미지 추가, 댓글 달기 — 는 중간 정도예요. 대부분의 브라우저 도구가 표준 폼 필드를 잘 처리해요. 예외는 Adobe LiveCycle로 만든 XFA 폼인데, 이건 Acrobat으로만 제대로 채울 수 있어요.

**텍스트 편집** — 위에 텍스트를 덮는 게 아니라 PDF 본문 텍스트 자체를 변경하는 것 — 이 진짜 어려운 부분이에요. PDF는 텍스트를 위치가 지정된 그래픽 글리프로 저장해요. 편집 가능한 문서 구조가 아니에요. 일부 도구가 이걸 지원한다고 주장하고, 실제로 되는 것도 있지만, 결과는 PDF 구성에 크게 달려 있어요.

## PDF24: 대부분의 작업은 여기서 시작하세요

두 계약서를 합쳐서 보내야 하거나, 스캔 문서가 20MB로 부풀어 압축이 필요할 때, [PDF24 Tools](/tool/tools-pdf24-org-en)가 첫 번째 선택지가 되어야 해요.

PDF24는 진짜 무료이고 회원가입 없이 사용할 수 있어요. 출력 파일에 워터마크 없고, 구독 유도를 위한 인위적인 일일 제한도 없고, 이메일도 요구하지 않아요. 기능 목록은 길어요. 병합, 분할, 압축, 회전, Word·Excel·JPG와의 상호 변환, 페이지 번호 추가, 비밀번호 보호, 비밀번호 제거, 스캔 텍스트 OCR 등. 독일 회사 geek software GmbH가 운영하며, 개인정보 처리방침에는 업로드된 파일이 처리 후 서버에서 삭제된다고 명시되어 있어요. 실제 문서를 업로드하는 사람에게는 중요한 세부 사항이에요.

인터페이스는 드래그 앤 드롭 방식이에요. 파일을 드롭하고, 도구가 처리하고, 결과를 다운로드해요. 가짜 진행 바도 없고, 「다운로드 준비 중」45초 카운트다운도 없고, 파일과 사용자 사이에 끼어드는 업셀 모달도 없어요. 당연한 것처럼 들리지만, 이걸 제대로 못 하는 사이트가 얼마나 많은지 놀랄 거예요.

PDF24가 잘 하는데 많이들 모르는 게 있어요. 압축 도구에서 압축 레벨을 선택할 수 있다는 거예요. 「권장」을 선택하면 눈에 띄는 품질 손실 없이 대부분의 파일을 40~60% 줄일 수 있어요. 「강력」은 더 압축하지만 이미지가 약간 흐려져요. 읽기용(인쇄가 아닌)으로 스캔 문서를 보낼 때는 「강력」이면 대개 충분해요. 대부분의 온라인 압축 도구는 선택지 없이 하나의 출력만 제공해요.

PDF24의 약점: PDF→Word 변환은 단순한 텍스트 문서엔 잘 되지만, 복잡한 레이아웃이나 텍스트와 표·그래픽이 혼재된 문서는 힘들어해요. 이건 모든 온라인 변환 도구의 공통 문제이지, PDF24 특유의 단점이 아니에요.

## TinyWow: PDF 이상이 필요할 때

[TinyWow](/tool/tinywow-com)는 같은 접근 방식 — 회원가입 불필요, 워터마크 없음, 무료 — 이지만, 도구 세트가 PDF를 넘어 동영상, 이미지, 파일 변환까지 확장돼요. 다양한 종류의 작업을 처리해야 할 때 유용합니다.

PDF 기능만 보면, TinyWow는 병합, 분할, 압축, 변환, 서명, 그리고 페이지 어디든 텍스트 박스·도형·이미지를 배치할 수 있는 「PDF Edit」 도구를 제공해요. 마지막 기능은 스캔된 플랫 양식 — 점선과 빈 칸은 있지만 실제 폼 필드는 없는 종류 — 을 작성할 때 특히 유용해요. 각 빈 칸 위에 텍스트 박스를 배치하고 입력하면 돼요. 우아하진 않지만 작동해요.

단점은 광고가 있다는 거예요. 광고는 있지만 압도적이지는 않아요. 광고 없는 환경을 원한다면 PDF24가 더 깔끔해요. 회원가입 없이 PDF와 다른 모든 형식을 한 사이트에서 처리하고 싶다면 TinyWow를 북마크해 두세요.

## 텍스트 편집과 양식 작성에 특화된 도구들

PDF의 기존 본문 텍스트를 변경해야 할 때 — 위에 덮는 게 아니라 이미 있는 내용을 편집하는 것 — [Sejda](https://www.sejda.com/)는 이걸 실제로 해내는 온라인 도구예요. 계정 없이 시간당 3개의 작업이 가능하고, 가끔씩 편집하는 용도로는 충분해요.

Sejda는 브라우저 기반 편집기에서 PDF를 열고, 기존 텍스트를 클릭해 인라인으로 수정할 수 있게 해줘요. 폰트 대체가 항상 완벽하지는 않아요(원본이 커스텀 임베디드 폰트를 사용했다면 Sejda가 비슷한 폰트로 대체해요). 하지만 표준 시스템 폰트로 만들어진 PDF라면 편집이 깔끔하게 돼요. 시간당 제한은 자동으로 초기화되고, 제한 내용도 투명해요 — Sejda는 제한을 작은 글씨에 숨기지 않아요.

양식 작성 전문으로는 — 정부 양식, 임대 신청서, 보험 서류 — [PDFescape](https://www.pdfescape.com/)가 2008년부터 로그인 없이 사용 가능했어요. 무료 티어는 10MB, 100페이지까지의 PDF를 지원해요. 적절한 폼 필드가 있는 문서는 클릭하고 입력하면 돼요. 플랫 스캔 양식의 경우, 빈 칸 위에 위치가 맞춰진 텍스트 박스를 추가하면 돼요. PDFescape는 주석도 지원해요. 하이라이트, 스티커 노트, 자유 그리기, 취소선 — 이메일로 답장을 보내기 전에 계약서를 검토하는 데 유용해요.

같은 PDF 양식을 반복해서 작성해야 하는 경우 — 경비 보고서 템플릿, 주간 타임시트 — 대부분의 온라인 도구는 세션 간에 작업 내용을 저장하지 않는다는 점을 알아두세요. 업로드하고, 작성하고, 다운로드하면, 도구는 모든 걸 잊어버려요. 사실 이건 개인정보 보호 기능이기도 해요. 세션 데이터도, 양식 내용도 저장되지 않아요. 하지만 매번 처음부터 시작해야 한다는 의미이기도 해요. 반복적으로 사용하는 양식이라면, 부분적으로 작성된 템플릿을 로컬에 저장해 두고 매번 그걸 업로드해서 출발점으로 삼는 게 실용적인 방법이에요.

## 빠른 참고: 어떤 도구를 써야 할까

| 작업 | 최적 도구 | 참고 사항 |
|------|---------|----------|
| PDF 병합 | PDF24 | 제한 없음, 워터마크 없음 |
| PDF 분할 | PDF24 또는 TinyWow | 둘 다 신뢰할 수 있음 |
| PDF 압축 | PDF24 | 크기 대 품질 균형이 좋음 |
| PDF → Word 변환 | PDF24 | 단순 문서에 효과적 |
| 표준 폼 필드 작성 | PDFescape | 대부분의 양식에 적합 |
| 텍스트 오버레이 추가 (플랫 양식) | TinyWow | 드래그앤드롭 텍스트 박스 |
| 기존 텍스트 편집 | Sejda | 시간당 3개 무료, 온라인 최선 선택 |
| 서명 이미지 추가 | TinyWow 또는 PDFescape | 손으로 그리기 또는 타이핑 |
| 스캔 PDF OCR | PDF24 | 스캔 텍스트를 검색 가능하게 |

## 프라이버시의 트레이드오프

위의 모든 도구는 처리를 위해 파일을 서버에 업로드해요. 일상적인 문서 — 임대 요약서, 뉴스레터 PDF, 컨퍼런스 일정표 — 라면 괜찮아요. 민감한 내용이 있다면 한 번 더 생각해 볼 필요가 있어요.

가장 프라이버시 친화적인 선택은 로컬 처리예요. PDF24의 데스크탑 앱(무료, 라이선스 키 없음, Windows용 제공)은 웹 버전이 하는 모든 것을 자신의 컴퓨터에서 처리해요. Linux 사용자를 위한 PDF24 CLI 도구도 있어요. 둘 다 계정 생성이 필요 없어요. 비교하자면, Adobe Acrobat은 오프라인 사용에도 구독과 Adobe 계정이 필요해요.

모든 무료 온라인 도구에 공통으로 적용되는 한 가지 제한: **개인정보 가리기(먹칠)는 신뢰할 수 없어요**. TinyWow, PDFescape, 또는 대부분의 브라우저 편집기에서 민감한 텍스트 위에 검은 박스를 그려도, 그 텍스트는 PDF의 기반 데이터에 여전히 존재해요 — 시각적으로 덮여 있을 뿐이에요. 적절한 도구를 가진 사람이라면 추출할 수 있어요. 진짜 먹칠 처리에는 파일 구조에서 내용을 영구적으로 제거하는 소프트웨어가 필요해요. Adobe Acrobat과 PDF24 데스크탑 버전은 이걸 제대로 처리해요. 숨겨진 내용이 실제로 중요한 상황 — 법적 문서, 개인 식별 정보가 있는 양식 — 에서는 온라인 도구로 먹칠하지 마세요.

다른 모든 경우에는: 보내기 전에 문서 병합하기, 첨부 파일 압축하기, 인쇄 전 양식 작성하기, 전화 통화 전 계약서에 주석 달기 — 노로그인 도구들이 충분히 작동해요. 구독이 필요 없어요. 계정도 필요 없어요. 브라우저에서 모두 처리할 수 있어요.

PDF 포맷은 30년이 넘었어요. Adobe에 돈 내지 않고 PDF를 다룰 수 있는 무료 도구들이 진짜로 따라잡았어요. 자신의 구체적인 작업에 맞는 도구를 선택하면, 대부분의 PDF 문제는 2분 안에 해결돼요.

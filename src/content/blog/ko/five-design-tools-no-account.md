Canva는 하나의 도구가 아니에요. 색상 팔레트 선택, 폰트 조합, 사진 배경 제거, 슬라이드 만들기, 플로차트 그리기 — 이렇게 다섯 가지 작업이 하나의 서비스에 묶여 있는 거예요. 각각의 작업에는 Canva보다 잘 처리하는 전용 도구가 있어요. 이메일 주소도 요구하지 않아요.

Canva 번들을 쓰면 편의성을 정체성으로 지불하게 돼요. 어떤 템플릿을 좋아하는지, 편집에 얼마나 시간을 쓰는지, 무엇을 내보내는지 — 매 세션마다 프로필이 쌓여요. 이 데이터를 활용해서 Canva는 Pro 구독을 유도하죠. 비즈니스 모델로는 이해할 수 있지만, 실제로 무언가를 교환하고 있는 거예요. 뭘 주고받는지 알아두는 게 좋아요.

아래 다섯 가지 도구는 그런 거래를 요구하지 않아요. 탭 열고, 작업하고, 닫으면 끝이에요. 계정 없음, 남의 서버에 저장되는 디자인 기록 없음, 요청한 적 없는 기능에 대한 마케팅 이메일도 없어요.

## Coolors: 60초 안에 색상 팔레트 완성

어디서부터 시작할지 모를 때 [Coolors](https://coolors.co)가 가장 빠른 방법이에요. 스페이스바를 누르면 5색 팔레트가 즉시 생성돼요. 한 번 더 누르면 또 새로운 조합이 나와요. 마음에 드는 색상은 잠가두고 나머지를 계속 랜덤 생성하다 보면 딱 맞는 조합이 나와요.

단순해 보이지만 Coolors는 꽤 깊이 있어요. 업로드한 사진에서 팔레트를 추출할 수도 있어요. 기존 이미지나 브랜드 에셋을 중심으로 디자인할 때 특히 유용해요. 특정 헥스 코드에서 시작해서 조화로운 옵션을 생성할 수도 있고요. 명도 대비 검사 기능도 내장되어 있어서 접근성 문제를 프로덕션에 들어가기 전에 잡아낼 수 있어요. WCAG 기준을 충족하지 못하는 팔레트는 잘 캘리브레이션된 모니터에서는 괜찮아 보여도 저시력 사용자에게는 사용 불가일 수 있거든요.

Canva에도 색상 도구가 있지만, 편집기 안에 들어가 있고 템플릿 시스템 중심으로 설계되어 있어요. Coolors는 오직 색상 선택 문제만을 위해 존재해요. 그 집중력이 확실히 느껴져요 — 무료 버전(회원가입 불필요)에서도 팔레트 생성, 랜덤화, 잠금 기능과 CSS, SCSS, PDF, PNG, SVG 형식 내보내기를 모두 사용할 수 있어요.

[nologin.tools의 Coolors 페이지](/tool/coolors-co)에서 계정 없이 사용할 수 있는 기능을 확인해 보세요.

## FontJoy: 뉴럴 네트워크가 추천하는 폰트 페어링

대부분의 사람들은 "전문적으로 보이는" 폰트를 고르거나 — 보통은 무난하고 안전한 걸 의미하죠 — 이전 프로젝트에서 쓰던 걸 그대로 사용해요. 어느 쪽도 흥미로운 결과를 내지 못하고, 다른 모든 디자인에 섞여 버려요.

[FontJoy](https://fontjoy.com)는 폰트 특성으로 훈련된 뉴럴 네트워크를 사용해서 제목, 소제목, 본문 텍스트 조합을 추천해 줘요. 사용하려는 폰트를 잠가두고 페어링을 계속 생성하다 보면 딱 맞는 걸 찾을 수 있어요. 프리뷰 텍스트가 실시간으로 업데이트되니까 폰트 이름만 비교하는 게 아니라 실제 타이포그래피 조합을 확인할 수 있어요.

이 모델이 최적화하는 건 유사성이 아니라 대비예요. FontJoy는 시각적으로 구분되면서도 조화를 이루는 폰트 — 절제된 본문 폰트와 함께 쓸 디스플레이 폰트 — 를 찾아요. 이게 실제로 효과 있는 페어링 원칙이에요. "비슷하게 생긴 폰트 찾기"와는 완전히 달라요. 후자는 단조로운 결과만 낳거든요.

계정 불필요해요. 결과물로 Google Fonts CSS 코드를 주니까 프로젝트에 바로 붙여넣을 수 있어요. Canva에서도 폰트를 둘러볼 수 있지만 페어링 지능은 없어요. 뭐가 잘 어울리는지는 스스로 판단해야 하는데, 그게 딱 어려운 부분이잖아요.

## remove.bg: 진짜 잘 되는 배경 제거

인물 사진이나 단순한 배경 앞에서 찍은 제품 이미지라면 [remove.bg](https://www.remove.bg)가 몇 초 만에 배경을 제거해 줘요. 수동 편집이 전혀 필요 없어요. 이미지를 업로드하고, 기다리고, 투명 PNG를 다운로드하면 끝이에요.

인물 피사체에 대한 정확도가 높아요. 배경 제거에서 역사적으로 가장 어려운 부분인 머리카락 가장자리도 잘 처리해요. 흰색이나 중립적인 배경의 제품 사진은 수동 수정 없이도 거의 완벽한 결과가 나오는 경우가 많아요.

Canva는 Pro 플랜에서만 배경 제거 기능을 제공해요. 월 약 15달러예요. 기능적 차이는 명확해요 — Canva 버전은 편집기 안에서, 지원되는 형식으로, 활성 구독이 있을 때만 작동해요. remove.bg는 업로드한 JPEG나 PNG에 작동하고 무료 플랜에는 계정도 필요 없어요. 단, 무료 버전은 저해상도 결과물을 제공해요. 전체 해상도는 유료 플랜이나 크레딧이 필요해요. 웹용 빠른 이미지라면 표준 해상도로 충분한 경우가 대부분이에요.

한 가지 알아둘 점 — remove.bg는 서버 측에서 이미지를 처리해요. 사진이 서버로 전송된다는 뜻이에요. 데이터 처리와 보관에 대한 [상세한 개인정보 처리방침](https://www.remove.bg/privacy)이 공개되어 있어요. 로컬 처리가 필요한 민감한 이미지라면 Rembg가 완전히 내 기기에서 실행되는 오픈소스 커맨드라인 대안이에요.

[nologin.tools의 remove.bg 페이지](/tool/remove-bg)에서 무료 버전 세부 사항을 확인해 보세요.

## Excalideck: 기업 PT 스타일 없는 프레젠테이션

완성도 높은 프레젠테이션 템플릿은 자신감과 확정감을 보여주고 싶을 때 유용해요. 하지만 아직 논의 중인 아이디어에 참여를 유도하고 싶을 때는 오히려 방해가 돼요. 세련된 Canva 덱을 들고 나타나면 참가자들은 그걸 이미 결론 난 것으로 받아들여요. 거친 것을 들고 나타나면 대화가 열린 채로 유지돼요.

[Excalideck](https://excalideck.com)은 Excalidraw의 손으로 그린 것 같은 캔버스 위에 슬라이드 내비게이션과 발표 모드를 더한 도구예요. 이 미적 감각은 의도적이에요 — 약간 불규칙한 도형, 손글씨 폰트, 렌더링된 게 아니라 스케치한 것처럼 보이는 화살표. 결과물은 "작업 중인 초안"이라는 메시지를 전달해요. 많은 기술 발표와 내부 계획 세션에서 이게 솔직한 메시지예요.

로그인 불필요라는 점은 특정 실용적인 문제를 해결해요. 낯선 기기 — 클라이언트 노트북, 공유 사무실 컴퓨터, 컨퍼런스 장소의 기기 — 에서 덱을 준비해야 할 때 Excalideck은 설정, 계정, 소프트웨어 설치 없이 모든 브라우저에서 열려요. Google Slides는 Google 계정이 필요해요. PowerPoint Online은 Microsoft 계정이 필요해요. Excalideck은 아무것도 필요 없어요.

시각적 완성도가 목표라면 Excalideck은 적합하지 않아요. 하지만 미적 감각보다 소통이 중요하거나, 청중이 아이디어를 질문하고 추가하도록 초대하고 싶다면 손으로 그린 스타일은 한계가 아니라 강점이에요.

[nologin.tools의 Excalideck 페이지](/tool/excalideck-com)에서 전체 기능 설명을 확인해 보세요.

## Diagrams.net: 박스와 화살표를 넘어선 실제 다이어그램

Canva는 화살표로 연결된 박스를 배치할 수 있어요. [Diagrams.net](https://app.diagrams.net)은 플로차트의 결정 노드가 마름모라는 것, 커넥터가 도형 가장자리에 스냅된다는 것, 레이아웃 알고리즘이 존재하는 이유, 엔티티-관계 다이어그램이 특정 규칙을 따른다는 걸 이해해요. 다이어그램 구조가 중요할 때 이 둘의 차이는 엄청나요.

Diagrams.net은 플로차트, UML 클래스 및 시퀀스 다이어그램, ER 다이어그램, 네트워크 아키텍처, 조직도 등을 지원해요. 도형 라이브러리가 크고 커넥터 라우팅이 스마트해요 — 선이 장애물을 피해 경로를 찾고, 직교 커넥터가 제대로 스냅되고, 요소를 이동해도 다이어그램이 엉망이 되지 않아요. 내보내기 옵션으로는 SVG, PNG, PDF, 그리고 네이티브 XML 형식이 있어요. XML은 사람이 읽을 수 있어 버전 관리와 잘 맞아요.

스토리지 모델은 개인정보를 중시하는 사용자에게 특히 좋아요 — 기본적으로 Diagrams.net은 로컬 파일에만 저장해요. Google Drive, OneDrive, GitHub에 연결할 수 있지만, 그건 내 선택이에요. 명시적인 동작 없이는 서버에 저장되지 않아요. XML 형식 덕분에 다이어그램이 설명하는 코드와 같은 문서 리포지토리에 파일을 함께 둘 수 있어요.

Canva의 다이어그램 기능은 간단한 시각적 설명에는 괜찮지만 그 이상은 아니에요. 다이어그램의 논리적 구조가 중요할 때 — 단지 일러스트하는 게 아니라 시스템을 그릴 때 — Diagrams.net이 맞는 도구예요.

## 이것들을 어떻게 함께 활용하면 될까요

[Canva 대안에 관한 이전 글](/blog/canva-alternatives-no-login)은 만드는 도구들을 다뤘어요 — 이미지 편집을 위한 Photopea, 기술 다이어그램을 위한 Excalidraw, 협업 화이트보딩을 위한 tldraw. 이 다섯 가지 도구는 지원 작업 — 무언가를 만들기 시작하기 전에 하는 의사 결정 — 을 다뤄요.

| 작업 | 로그인 불필요 도구 | Canva 비교 |
|------|--------------|------------------|
| 색상 팔레트 | [Coolors](https://coolors.co) | 편집기 안에 묻혀 있고 템플릿 중심 |
| 폰트 페어링 | [FontJoy](https://fontjoy.com) | 페어링 지능 없음 |
| 배경 제거 | [remove.bg](https://remove.bg) | Pro 플랜만 (~월 $15) |
| 프레젠테이션 | [Excalideck](https://excalideck.com) | 템플릿 기반, 계정 필요 |
| 다이어그램 | [Diagrams.net](https://app.diagrams.net) | 기본 박스-화살표만 가능 |

모두 회원가입 불필요해요. 동의 없이 작업물이 저장되지 않아요. 모두 플러그인이나 설치 없이 표준 브라우저에서 열려요.

> Canva가 하는 모든 걸 계정 없이 할 수 있는 방법은 없어요. 하지만 Canva가 못하는 모든 걸 계정 없이 할 수 있는 방법은 있어요.

이 도구들이 계정 유지라는 안전망 없이도 살아남을 수 있는 건 다른 수익 모델 — 선택적 유료 티어, 오픈소스 기여, 최소한의 광고 — 로 구축되어 있기 때문이에요. 계정 없이도 진짜 유용해야 한다는 압박이 관성이 아닌 실력으로 사용자를 계속 붙잡는 소프트웨어를 만드는 경향이 있어요. [nologin.tools 디렉토리](/tool/nologin-tools)에는 수십 개 카테고리에 걸쳐 회원가입 없이 작동하는 게 확인된 도구들이 더 많이 있어요.

이 도구들을 배우는 가장 좋은 이유는 정확히 개인정보 보호는 아니에요 — 그것도 중요하지만. 특정 작업에 특화된 도구는 모든 걸 적당히 하는 번들 제품보다 그 작업에서 거의 항상 더 나아요. Canva의 폰트 선택기는 FontJoy만큼 똑똑해질 수 없을 거예요. 배경 제거 기능은 유료 기능으로 남아 있을 거예요. 다이어그램 도구는 기본적인 수준을 벗어나지 못할 거예요. 특화된 로그인 불필요 도구는 더 좁은 작업 범위를 갖지만, 그걸 더 잘 해요.

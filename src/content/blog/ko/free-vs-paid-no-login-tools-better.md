유료 소프트웨어에는 이미지 문제가 생겼어요. 제품이 나빠서가 아니라, "유료 = 좋은 것"이라는 공식이 한때는 믿을 만했기 때문이에요. 돈을 내면 그만큼의 가치를 얻었어요. 그런데 그 약속이 조용히 무너지고 있어요.

변화는 SaaS 가격 모델이 일회성 구매에서 월정액 구독으로 옮겨가면서 시작됐어요. Adobe Creative Suite는 700달러짜리 패키지에서 월 60달러 구독으로 바뀌었어요. Notion은 가격 구조를 올렸고, Zoom의 무료 플랜은 축소됐어요. 그사이 브라우저 기반 도구들 — 개인 개발자나 소규모 팀이 만들고, 완전히 클라이언트 사이드에서 실행되며, 계정도 필요 없는 — 이 대부분의 사람들이 눈치채기도 전에 빠르게 따라잡았어요.

"무료 vs 유료?"라는 질문에 솔직하게 답하자면: 작업에 따라 달라요. 하지만 로그인 불필요 도구가 단순히 "그럭저럭 쓸 만한" 수준을 넘어서, 실제로 더 나은 특정 영역들이 있어요.

## 유료 도구가 광고하지 않는 프라이버시 격차

기능을 비교하기 전에, 유료 도구가 그 기능을 제공하기 위해 무엇을 수집하는지 먼저 생각해봐야 해요. 대부분의 구독 소프트웨어는 파일, 기록, 설정, 사용 패턴을 계정에 연결해요. 그 계정은 수익화 가능한 데이터예요. 제품 개선에 쓰이기도 하지만, 이탈 방지 모델링, 타겟 업셀링, 그리고 경우에 따라서는 제3자와의 공유에도 활용돼요.

브라우저에서 완전히 실행되는 로그인 불필요 도구는 여러분의 데이터를 여러분 기기에서 처리해요. 어떤 것도 컴퓨터 밖으로 나가지 않아요. 해킹당할 계정도, 구축될 프로필도, 걱정할 구독 이메일 목록도 없어요.

이건 민감한 작업에서 특히 중요해요. 법률 문서, 의료 이미지, 재무 스프레드시트, 개인 사진 같은 것들이요. "이 파일에 누가 접근할 수 있나요?"라는 질문의 답이 "여러분의 브라우저뿐이에요"가 될 때, 그 답은 훨씬 명쾌해요. [전자 프론티어 재단](https://www.eff.org/issues/privacy)은 "프라이버시를 중시한다"고 마케팅하는 유료 서비스들조차 실제로는 광고보다 훨씬 많은 데이터를 수집한다는 걸 문서로 남겨왔어요.

데이터를 서버에 전혀 보내지 않기 때문에 수집 자체가 불가능한 도구는, 카테고리 자체가 다른 프라이버시 이야기예요. 대부분의 유료 도구 가격 페이지에서는 찾아볼 수 없는 기능이죠.

## 이미지 편집: 격차가 완전히 좁혀진 분야

Photoshop의 대부분의 역사에서, 레이어 이미지 편집을 위한 브라우저 대안은 존재하지 않았어요. PSD 파일은 곧 Adobe 소프트웨어를 의미했어요. 그런데 이제 달라졌어요.

[Photopea](https://www.photopea.com)는 설치도, 계정도 없이 PSD, AI, XCF, Sketch 파일을 브라우저에서 직접 열 수 있어요. 레이어, 마스크, 블렌드 모드, 스마트 오브젝트, 조정 레이어, 그리고 일반적인 Photoshop 작업에서 쓰는 대부분의 도구를 지원해요. 개발자에 따르면 월 1,000만 명 이상이 사용한다고 하는데, 이 숫자는 바이럴 화제성이 아닌 진짜 유용함을 반영해요.

Adobe Photoshop은 단독 앱 기준 월 23.99달러 (Creative Cloud 포함 시 월 54.99달러)예요. 한 달에 한 번 PSD를 편집하거나, 소프트웨어 설치 없이 클라이언트의 레이어 파일을 열어야 하는 사람에게는 정당화하기 어려운 구독료예요. [Photopea](/tool/photopea-com)는 계정 없이 이걸 처리하고, 5초 안에 로드돼요.

Photopea가 부족한 점은 어디일까요? Generative Fill, Neural Filters, 그리고 Photoshop의 AI 기반 선택 도구들은 유료 버전이 확실히 뛰어나요. 이런 기능을 매일 쓰는 분이라면 구독이 합리적이에요. 하지만 대부분의 사람들이 실제로 Photoshop을 쓰는 용도인 크기 조정, 보정, 합성, 레이어 편집에서는 — Photopea가 진짜 대안이에요.

이미지 압축도 비슷한 이야기를 해요. Google의 브라우저 기반 이미지 최적화 도구인 [Squoosh](/tool/squoosh-app)는 WebP, AVIF, MozJPEG, OxiPNG를 지원해요. WebAssembly를 통해 로컬에서 실행되기 때문에 파일이 서버로 전송되지 않아요. 유료 서비스들은 건당 또는 월정액을 청구해요. 대부분의 압축 작업에서 Squoosh는 비슷한 결과를 내면서 데이터가 기기를 떠나지 않아요. 유료 대안들은 이에 대한 설득력 있는 반론을 내놓기 어려워요.

## 개발자 도구: 이미 무료가 이긴 영역

개발자 도구 카테고리는 로그인 불필요 도구가 가장 명확하게 앞서 있는 분야예요 — 몇몇 경우에는 비교 자체가 의미 없을 정도예요. 몇 년 전부터 그랬고요.

| 작업 | 로그인 불필요 도구 | 유료 경쟁 도구 |
|------|--------------|-----------------|
| Regex 테스트 | [Regex101](/tool/regex101-com) | Sublime Text 플러그인 ($99), 에디터 애드온 |
| SQL 플레이그라운드 | [DB Fiddle](/tool/db-fiddle-com) | DataGrip ($9.90/월) |
| 코드 스크린샷 | [Carbon](/tool/carbon-now-sh) 또는 [Ray.so](/tool/ray-so) | CleanShot X ($29) |
| API 테스트 | [Hoppscotch](/tool/hoppscotch-io) | Postman Pro ($12/월) |
| JSON 시각화 | [JSON Crack](/tool/jsoncrack-com) | 다양한 유료 도구 |
| SVG 최적화 | [SVGOMG](/tool/jakearchibald-github-io-svgomg) | ImageOptim ($24.99) |

[Regex101](https://regex101.com)은 10년 넘게 Regex 테스트의 표준이 되어왔어요. 계정도, 제한도 없고, PCRE, Python, JavaScript, Go 정규식 엔진을 지원하며 실시간 매칭, 캡처 그룹 검사, 커뮤니티 패턴 라이브러리를 제공해요. 이보다 더 잘 하는 유료 대안을 떠올리기가 어려워요.

[Hoppscotch](/tool/hoppscotch-io)는 오픈소스이고, 회원가입 없이 브라우저에서 실행되며, REST, GraphQL, WebSocket, Socket.IO 테스트를 지원해요. Postman은 최근 핵심 협업 기능들을 로그인 장벽 뒤로 옮기고, 클라우드 연결 모델로 사용자를 유도하고 있어요. Hoppscotch는 그렇게 하지 않았어요. 빠른 API 테스트 — 요청 보내기, 헤더 확인, 응답 본문 검사 — 를 위해서라면 Hoppscotch는 더 빠르게 열리고, 작업을 시작하는 데 인증이 필요하지 않아요.

패턴은 일관돼요. 브라우저 사용을 위해 만들어진 개발자 도구들은 더 빠르게 열리고, 마찰이 적으며, 보호해야 할 서버 사이드 상태가 없기 때문에 계정을 요구하지 않아요. 작업은 로컬에서 이루어지고, 결과는 즉시 나타나며, 동의하지 않은 클라우드에 아무것도 남지 않아요.

## 데이터 수집 없는 파일 공유와 커뮤니케이션

Zoom의 2020년 논란들 — 예상치 못한 인프라를 통한 통화 라우팅 가능성, "종단간 암호화"가 사실은 전송 암호화였다는 것 — 은 사람들이 실제로 회의 프라이버시에 얼마나 신경 쓰는지에 대한 스트레스 테스트였어요. 많은 사람들이 잠깐 다른 곳으로 갔다가 다시 돌아왔어요.

[Jitsi Meet](https://meet.jit.si)은 구조적으로 다른 무언가를 제공해요. 완전한 오픈소스 (Apache 2.0 라이선스), 자체 호스팅 가능, Insertable Streams 구현을 통한 실제 종단간 암호화를 지원해요. 회의를 시작하거나 참가하는 데 계정이 필요 없어요. meet.jit.si의 공개 인스턴스는 브라우저에서 즉시 작동해요.

Zoom의 무료 플랜은 이제 그룹 회의를 40분으로 제한해요. Jitsi Meet는 공개 인스턴스에서 시간 제한이 없어요. 소규모 팀이나 가끔 통화하는 경우, 이건 의미 있는 차이예요 — 비용 면에서도, 대화 데이터가 어떻게 처리되는지 면에서도요.

일회성 파일 전송에는 [Wormhole](/tool/wormhole-app)이 종단간 암호화로 최대 10GB 파일을 보낼 수 있어요. 파일은 24시간 후 또는 첫 다운로드 후 자동 삭제돼요. 계정도, 가입도, 등록도 필요 없어요. WeTransfer Plus ($16/월) 같은 유료 대안은 영구 저장소와 더 긴 링크 만료 기간을 추가해요 — 파일을 반복적으로 공유하는 워크플로우에서는 합당한 장점이에요. 하지만 파일을 한 번 보내는 경우에는, Wormhole이 더 프라이버시 친화적이고, 더 큰 파일을 처리하며, 설계상 흔적을 남기지 않아요.

이런 도구들의 프라이버시 친화적인 특성은 마케팅 기능으로 덧붙여진 게 아니라 아키텍처에 내재되어 있어요. 그건 의미 있는 차이예요.

## 유료 도구가 진짜로 이기는 경우

이 비교는 솔직할 때만 유용해요. 유료 도구들은 중요한 특정 영역에서 실제 장점이 있어요.

**지속적인 협업**이 가장 명확한 경우예요. Figma의 멀티플레이어 편집, Notion의 공유 데이터베이스, GitHub의 코드 리뷰 워크플로우는 여러 사람이 같은 상태에 동시에, 비동기적으로 접근해야 한다는 가정 위에 만들어졌어요. 브라우저 기반 로그인 불필요 도구들은 대부분 단일 세션이에요. 10명의 팀이 2주에 걸쳐 같은 문서를 편집해야 한다면, 구독료는 무료 대안에 실제로 존재하지 않는 조율 인프라를 위해 쓰이는 거예요.

**신뢰할 수 있는 지원과 가동 시간 보장**은 전문적인 맥락에서 중요해요. 유료 서비스가 마감 전에 다운되면, 계약과 지원 채널이 있어요. 로그인 불필요 도구들은 종종 공식 SLA 없이 소규모 팀이나 개인이 유지 관리해요.

**세련된 모바일 경험**은 여전히 격차가 있어요. 많은 브라우저 기반 도구들이 모바일에서 그럭저럭 작동하지만, iA Writer, Bear, Ulysses 같은 앱들은 브라우저 에디터가 따라가지 못한 방식으로 모바일 글쓰기 경험을 다듬었어요. 폰이나 태블릿에서 진지하게 작업한다면, 이게 종종 결정적 요소가 돼요.

**고급 AI 통합**은 유료 제품들을 점점 더 차별화하고 있어요. Adobe Firefly의 Generative Fill, GitHub Copilot, Notion AI는 오픈 브라우저 대안들이 아직 따라가지 못한 것들을 하고 있어요. 문제는 그 기능을 월 비용이 합리적일 만큼 자주 사용하느냐예요.

## 대부분의 사람들이 건너뛰는 계산

SaaS 업계는 사람들에게 "유료 = 진지함"이라는 인식을 심어왔어요. 구독은 전문적인 느낌을 줘요. 믿음직해 보여요. 브라우저 기술이 제한적이고 웹앱이 실제 작업을 할 수 없었던 시절에는 그 직관이 말이 됐어요.

[WebAssembly](https://webassembly.org)가 브라우저에서 실행할 수 있는 것을 바꿔놓았어요. 클라이언트 사이드 처리는 파일이 로컬에 머물게 해요. 오픈소스 개발은 상업적 동등물을 기능 면에서 따라잡고, 종종 프라이버시 면에서는 앞서는 도구들을 만들어냈어요.

해볼 만한 계산은 간단해요. 이 도구가 돈, 프라이버시, 마찰 면에서 얼마나 드나요? 때로는 구독이 그 계산에서 분명히 이겨요. 하지만 자주는 아니에요 — 특히 가끔씩 하는 작업, 로컬에 보관하고 싶은 파일, 지금 당장 작동하는 무언가가 필요한 워크플로우에서는요.

[nologin.tools](/tool/nologin-tools)에 나열된 도구들은 계정 없이 작동하는 게 검증되어 있어요. 카테고리별로 둘러보세요 — 이미지 편집, 개발자 유틸리티, 파일 공유, 계산기. 매월 비용을 내고 있던 워크플로우의 단계에, 같은 일을 하고 데이터를 로컬에 보관하며 이메일 주소가 없어도 시작할 수 있는 브라우저 대안이 있다는 걸 발견할 수도 있어요.

질문은 한 번도 "무료냐 유료냐?"가 아니었어요. 더 나은 질문은 이거예요. 실제로 무엇에 돈을 내고 있고, 그게 그만한 가치가 있나요?

付费软件现在有点名誉危机。倒不是说它变差了，而是以前"付费 = 值得付"这个逻辑一直很稳，但这个默契已经悄悄瓦解了。

转折点发生在 SaaS 定价从买断制变成月订阅制之后。Adobe Creative Suite 从一次性 700 美元变成了每月 60 美元，Notion 不断提高收费门槛，Zoom 也缩减了免费额度。与此同时，那些跑在浏览器里、不需要注册账号的工具——很多是个人开发者或小团队做的——进化速度比大多数人意识到的要快得多。

"免费还是付费？"这个问题的诚实答案是：看具体任务。但有些特定场景里，免登录工具不只是"够用"，而是真的更好。

## 付费工具不会主动告诉你的隐私差距

在比较功能之前，先想想付费工具为了提供这些功能都在收集什么。大多数订阅制软件会把你的文件、历史记录、偏好设置和使用习惯全部绑定到账号上。这个账号本身就是可变现的数据，用来改进产品没问题——但也会被用于留存分析、精准追加销售，有时候还会和第三方共享。

完全跑在浏览器里的免登录工具，数据处理都在你的设备上完成，任何东西都不会离开你的电脑。没有账号可以被泄露，没有用户画像需要维护，也不用担心订阅邮件列表的问题。

这一点在处理敏感内容时尤其重要：法律文件、医疗影像、财务表格、私人照片。当答案是"只有你的浏览器能访问这个文件"时，"谁能看到这个文件？"这个问题就有了一个清晰得多的答案。[电子前哨基金会](https://www.eff.org/issues/privacy)记录了很多案例，即便是主打"隐私友好"的付费服务，实际收集的数据也远超其营销材料所暗示的范围。

一个工具因为从不接触服务器而根本无法收集你的数据，这从根本上就是不同性质的隐私保障。这种特性你在大多数付费工具的定价页面上是找不到的。

## 图像编辑：差距已经完全弥合

在 Photoshop 的大部分历史里，浏览器里没有真正的分层图像编辑替代品。PSD 文件等于 Adobe 软件，没有例外。这个局面已经改变了。

[Photopea](https://www.photopea.com) 可以直接在浏览器里打开 PSD、AI、XCF 和 Sketch 文件，无需安装，无需账号。它支持图层、蒙版、混合模式、智能对象、调整图层，以及大多数日常 Photoshop 操作中会用到的工具。开发者公布的月活用户超过 1000 万——这个数字反映的是真实的实用价值，不是一时的病毒式传播。

Adobe Photoshop 单独购买是每月 23.99 美元（Creative Cloud 套餐是每月 54.99 美元）。对于一个月只需要编辑一次 PSD、或者想打开客户提供的分层文件但不想安装软件的人来说，这个订阅费很难说值得。[Photopea](/tool/photopea-com) 不需要账号，五秒内就能加载完成。

Photopea 的短板在哪？创意填充、神经网络滤镜，以及 Photoshop 那些 AI 驱动的选区工具，付费版本确实更强。如果你的工作每天都依赖这些功能，订阅是合理的。但对于调整大小、修图、合成和基于图层的编辑——也就是大多数人实际使用 Photoshop 的场景——Photopea 是真正的替代方案。

图像压缩的情况类似。Google 的浏览器端图像优化工具 [Squoosh](/tool/squoosh-app) 支持 WebP、AVIF、MozJPEG 和 OxiPNG，通过 WebAssembly 在本地运行，文件不会上传到服务器。付费服务按次收费或按月收费。对于大多数压缩需求，Squoosh 的效果相当，而且数据零上传。付费的那些竞品对此很难给出有说服力的反驳。

## 开发者工具：免费早就赢了

开发者工具这个类别，免登录工具最为领先——某些方面甚至领先多年，差距根本不是一个量级的。

| 任务 | 免登录工具 | 付费竞品 |
|------|-----------|---------|
| 正则测试 | [Regex101](/tool/regex101-com) | Sublime Text 插件（99 美元），编辑器附加组件 |
| SQL 演练 | [DB Fiddle](/tool/db-fiddle-com) | DataGrip（9.90 美元/月） |
| 代码截图 | [Carbon](/tool/carbon-now-sh) 或 [Ray.so](/tool/ray-so) | CleanShot X（29 美元） |
| API 测试 | [Hoppscotch](/tool/hoppscotch-io) | Postman Pro（12 美元/月） |
| JSON 可视化 | [JSON Crack](/tool/jsoncrack-com) | 各种付费工具 |
| SVG 优化 | [SVGOMG](/tool/jakearchibald-github-io-svgomg) | ImageOptim（24.99 美元） |

[Regex101](https://regex101.com) 十多年来一直是正则测试的标准工具。不需要账号，没有使用限制，支持 PCRE、Python、JavaScript 和 Go 的正则引擎，提供实时匹配、捕获组检查和社区模式库。你很难找到一个付费替代品在这方面做得更好。

[Hoppscotch](/tool/hoppscotch-io) 是开源的，不需要注册就能在浏览器里运行，支持 REST、GraphQL、WebSocket 和 Socket.IO 测试。Postman 最近把关键的协作功能移到了登录墙后面，并推动用户接入其云端模式。Hoppscotch 没有这么做。对于快速 API 测试——发一个请求、查看响应头、检查响应体——Hoppscotch 打开更快，不需要认证就能开始工作。

这个规律是一致的：为浏览器使用而构建的开发者工具，打开更快、摩擦更少，也不需要账号——因为没有需要保护的服务器端状态。工作在本地完成，结果立即呈现，也没有什么东西会同步到你未曾同意的某个云端。

## 文件共享与通信：不被数据收集绑架

Zoom 在 2020 年的那些争议——通话可能被路由到意料之外的基础设施、所谓的"端对端加密"其实只是传输层加密——是对人们究竟有多在意会议隐私的一次压力测试。很多人短暂地切换到了其他工具，然后又慢慢漂回去了。

[Jitsi Meet](https://meet.jit.si) 在结构上提供了不同的东西。它完全开源（Apache 2.0 许可），支持自部署，通过其 Insertable Streams 实现支持真正的端对端加密。不需要账号就能发起或加入会议，公共实例 meet.jit.si 在浏览器里立即可用。

Zoom 的免费版现在把群组会议限制在 40 分钟。Jitsi Meet 在公共实例上没有时间限制。对于小团队或偶尔通话的场景，这是个实质性的差别——无论是费用上的，还是你的通话数据去向上的。

一次性文件传输方面，[Wormhole](/tool/wormhole-app) 支持端对端加密地发送最大 10 GB 的文件，24 小时后或首次下载后自动删除，无需账号，无需注册，什么都不用填。付费替代品如 WeTransfer Plus（16 美元/月）提供持久存储和更长的链接有效期——对于需要反复共享文件的工作流来说确实是合理优势——但如果只是发一次文件，Wormhole 更私密，支持更大的文件，而且设计上就不留痕迹。

这些工具的隐私友好特性是内建于架构中的，不是作为营销功能后来加上去的。这个区别是有实质意义的。

## 付费工具真正赢的地方

这个比较只有诚实才有价值。付费工具在某些特定方面确实有真实的优势。

**持久协作**是最明显的场景。Figma 的多人实时编辑、Notion 的共享数据库、GitHub 的代码审查工作流，都建立在"多人需要同时异步访问同一状态"的假设上。基于浏览器的免登录工具基本上是单会话的。当一个十人团队需要在两周内编辑同一份文档时，订阅费买的是真正不存在于免费替代品中的协调基础设施。

**可靠的支持和正常运行时间保障**在专业场景中很重要。如果付费服务在截止日期前挂了，你有合同和支持渠道。免登录工具通常由小团队或个人维护，没有正式的 SLA。

**完善的移动端体验**仍然存在差距。很多浏览器端工具在移动端也能用，但像 iA Writer、Bear、Ulysses 这样的 App 对移动端写作体验的打磨，是浏览器编辑器还没复制出来的。如果你经常在手机或平板上认真工作，这往往是个决定性因素。

**深度 AI 集成**正在越来越成为付费产品的差异化卖点。Adobe Firefly 的创意填充、GitHub Copilot、Notion AI，都在做开放浏览器替代品还没跟上的事情。问题是你是否足够频繁地使用这些功能，让每月的费用变得合理。

## 大多数人跳过的那道计算题

SaaS 行业训练了用户把"付费"和"专业"画等号。订阅感觉正式，感觉可靠。当浏览器技术还很受限、Web 应用还做不了真正工作的时候，这个直觉是有道理的。

[WebAssembly](https://webassembly.org) 改变了浏览器能执行什么。客户端处理意味着你的文件留在本地。开源开发产出了在功能上可以与商业产品媲美的工具，在隐私方面往往还要超过它们。

值得做的计算其实很简单：这个工具在金钱上、在隐私上、在使用摩擦上分别要花什么代价？有时候订阅在这道题里明显胜出。更多时候并不——尤其是偶尔才做的任务、你更愿意在本地处理的文件，或者那些你只需要"现在能用就行"的工作流。

[nologin.tools](/tool/nologin-tools) 上列出的工具都经过验证，无需账号即可使用。按分类浏览——图像编辑、开发工具、文件共享、计算器。你可能会发现，工作流里那个你每月在付费的步骤，有一个浏览器替代品能完成同样的工作，数据留在本地，也不需要你的邮件地址就能开始。

问题从来不是"免费还是付费？"更好的问题是：你究竟在为什么付钱，这钱花得值吗？

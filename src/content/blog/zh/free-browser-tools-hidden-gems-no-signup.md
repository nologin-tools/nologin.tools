大多数浏览器工具找到自己受众的方式千篇一律：一条病毒式推文，一次 Reddit 首页上榜，然后每个人都开始收藏那几款耳熟能详的应用。错过那个窗口的工具就此沉寂——哪怕它们非常出色，哪怕它们已经默默运转了多年。

本文要介绍六款完全不需要账号的免费浏览器工具。不用邮箱，不用注册，不用记密码。有些是近期新品，有些已存在多年却从未被广泛关注。它们都能做些真正有用的事，而且在你开始使用之前不会要求你注册。

## 「无需登录」在这里究竟意味着什么

这个词其实有光谱之别。有些工具打着「无需登录」的旗号，却把有用的功能锁在注册墙后，或者 24 小时后让你的工作消失，以此逼你创建账号。这个列表不是那种类型。

这里的每一款工具在不注册的情况下都完全可用——你看到的就是你第一次加载时得到的。其中几款完全在你的浏览器中运行，这意味着你的数据根本不会到达服务器。这既是隐私上的好事，也意味着这些工具可以离线使用、秒开、而且不会因为某家公司转型或关门而失效。

如果你想看更广泛的经过验证的免登录工具列表，[nologin.tools](/tool/nologin-tools) 收录了各类别的数百款工具。但以下六款值得专门介绍。

## led.run — 把任何屏幕变成滚动显示屏

当你需要在屏幕上显示滚动文字——在音乐会、工作坊，或者接待台的显示器——大多数方案要么需要专用硬件，要么是需要注册的应用。[led.run](/tool/led-run) 是一款免费的浏览器工具，两者都不需要。

打开页面，输入文字，你的浏览器窗口就变成了全屏滚动的 LED 风格文字跑马灯。你可以调整字体大小、滚动速度、文字颜色和方向。渲染效果清晰、高对比度，在房间另一端也清晰可读。

听起来很小众。但如果你有过在活动前临时需要拼凑点什么的经历，这种工具发一个链接给别人就能用，对方会立刻收藏的。无需安装，无需注册，无需下载。

## tmp.tf — 最简单的跨设备剪贴板

场景是这样的：你在工作电脑上，需要把一段文字——一个网址、一段代码、一个电话号码——传到手机或另一台电脑上。没有配置好的共享剪贴板应用，发邮件给自己又觉得太蠢。

[tmp.tf](/tool/tmp-tf) 是一个基于浏览器的临时剪贴板。粘贴文字，获得一个短链接，在另一台设备上打开这个链接，内容就在那里了。把它想象成一次性用途的 pastebin：默认临时，没有历史记录，不需要账号。

它区别于普通 pastebin 的地方在于设计意图。这不是用来发布代码片段或公开分享内容的工具——它专为快速个人同步而生。URL 就是整个工作流程。用完关掉，忘了它。

## Datasette Lite — 在浏览器里对自己的文件跑 SQL 查询

如果你曾经有过 CSV 或 SQLite 文件，想在不搭建数据库服务器的情况下对它跑查询，Datasette Lite 值得了解。它完全在你的浏览器中运行 [Datasette](https://datasette.io/)——Simon Willison 的强大数据探索工具——使用 [WebAssembly](https://developer.mozilla.org/en-US/docs/WebAssembly)。无需安装，无需后端，无需账号。

你从本地加载一个 SQLite 数据库文件，完整的 Datasette 界面就出现了：浏览表格、分面过滤、SQL 查询执行、数据导出。你的数据永远不会离开浏览器。SQLite 引擎直接编译为 WebAssembly，意味着一个生产级数据库完全在你的设备上运行。

对于需要快速探索本地数据集的数据记者、分析师或开发者来说，这很重要。大多数类似工具要么需要服务器搭建，要么把你推向云端存储和账号注册。[Datasette Lite](/tool/lite-datasette-io) 两者都不需要。

> 「最好的工具是那些让你忘了自己在使用基础设施的工具。」

这句话在这里非常适用。本来需要多步搭建的事——安装 SQLite、载入数据库、配置 Datasette、开启服务器——变成了打开一个浏览器标签页。

## Markmap — 把 Markdown 大纲变成思维导图

写结构化笔记很有用，但把它们看成一张相互连接的图是另一种体验。[Markmap](/tool/markmap-js-org) 能将 Markdown 大纲——标题、子标题、嵌套列表——实时转换成可交互、可缩放的思维导图。无需拖拽，无需手动摆放节点，无需注册。

操作方式：在左侧面板写 Markdown，思维导图在右侧自动生成。你可以展开或折叠分支、缩放和平移，并将结果导出为 SVG。特别适合把文档结构变成可以在演示中展示或作为可视概览分享的东西。

不像 MindMeister 或 Miro 之类的思维导图工具——你还没做任何事就被推着去注册——Markmap 是开源的，从第一次访问起就完全可用。这种约束是有意为之的：输入就是文本，所以你在用自己已经熟悉的格式工作。

Markmap 也是回答「免登录工具为何重要，不只是因为方便」这个问题的好例子。当一个工具完全在浏览器中运行，并以文本形式管理其状态时，就没有账号会丢失，不需要在订阅到期前导出，没有锁定。内容从一开始就是你的。

## Goblin.tools — 无需账号的 AI 任务拆解

标准待办软件假设你已经知道如何把任务拆分成步骤。对很多人来说——尤其是有 ADHD 或其他执行功能障碍的人——这个假设本身就是全部问题所在。[Goblin.tools](/tool/goblin-tools) 建立在一个不同的模型上：你描述需要做什么，AI 来帮你拆解。

核心功能叫「Magic To-Do」，接收一个模糊的任务描述（「准备演示文稿」「回复那些邮件」）并生成具体的分步清单。你可以调整粒度——对感觉难以开始的任务多几步，对只需要一个起点的任务少几步。还有另外一个估算任务时长的工具、一个把文字块转换成清单的工具，以及一个分析书面沟通语气的工具。

值得注意的是，核心工具无需注册即可使用。AI 驱动的功能通常有账号墙，既为限速也为数据收集。Goblin.tools 是个例外。你可以使用主要的任务拆解工具、收藏一个会话，随时回来——全程不需要留下任何邮箱地址。

这是专注于一个狭窄领域，做得很好。这是这个列表中大多数工具共同的模式。

## SiteAge — 查一个网站是什么时候诞生的

评估一个不熟悉的网站——一个新闻来源、一个供应商、一个你即将用来完成任务的工具——往往意味着你想知道它有多老。它存在多久了？什么时候第一次被收录？它有没有发生过剧烈变化？

[SiteAge](/tool/siteage-org) 查询 [互联网档案馆的 Wayback Machine](https://web.archive.org/)，返回一个网站的大致建立日期以及它多年来外观变化的时间线。输入一个 URL，你就能看到它的历史。无需注册，无需 API 密钥。

这是一款只做一件事的小工具，但它填补了一个真实的空白。Wayback Machine 有这些数据；SiteAge 让你无需手动翻历史快照就能访问这些数据。对于研究人员、记者或任何正在对陌生域名做尽职调查的人来说，这是一步就能完成的捷径。

它不需要创建账号这一点与它的用途非常一致。一个用来评估其他网站的工具，本身不应该在这个过程中收集你的数据。

## 为什么免登录工具越来越好

这些工具存在有一个结构性原因：WebAssembly 让在浏览器中运行正儿八经的软件——编译器、数据库、图像处理器——成为可能。过去需要服务器的事情现在可以在本地运行。这改变了工具构建的经济学，因为纯客户端工具没有持续的基础设施成本，也就没有通过账号和数据来变现的压力。

开源社区充分利用了这一点。Datasette Lite 和 Markmap 这样的工具由开发者维护，是因为这些工具有用，而不是因为它们是某个 SaaS 漏斗的一部分。这种激励一致性产生了真正尊重用户的软件。

这个列表中不是每款工具都是开源的，但「先让你用，不要求任何东西」的无登录原则贯穿所有工具。这些工具信任你来使用它们，不需要知道你是谁。

如果你发现自己经常使用这些工具中的某一款，不妨去看看[完整目录](/tool/nologin-tools)，找找同类别中的其他工具。类似的好东西还有很多。

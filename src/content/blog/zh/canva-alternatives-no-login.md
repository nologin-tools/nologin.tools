关于 Canva 免费版，有一件事要说清楚：它确实很好用。这让这篇文章写起来有点尴尬。Canva 界面精良、速度快，还有数千个模板，让不擅长设计的人也能做出像样的作品。但它需要账号。而这个账号要求背后做的事情远比表面看起来多。

当你注册时，Canva 获得了你的邮箱、你的设计历史、你的 IP 地址、你的使用习惯，以及一个向你推销每月 15 美元订阅的直接渠道。这就是那笔交易——你的注意力和数据为免费层级买单。大多数人接受这笔交易时甚至没有多想。你可能也觉得没问题。但值得知道的是，替代品确实存在，因为那些完全无需注册即可在浏览器中使用的设计工具，已经变得相当出色。

Canva 的登录要求也会在你意想不到的具体情况下出现：共用的办公电脑、客户的笔记本、你平时不用的设备上的快速任务，或者你宁愿不让企业 SaaS 供应商记录你在设计什么的敏感工作。还有账号泛滥问题。如果你使用十几个网络工具，每个都需要注册，那么你现在就有十几个营销邮件订阅需要最终退订，你的邮箱地址就存在于十几个地方。

这五款无需登录的工具让你打开标签页、完成创作、导出文件、关闭标签页。没有邮件确认，没有引导清单，没有"你本月已用完 5 次免费导出中的 3 次"的提示。

## Photopea：比你想象的更接近 Photoshop

[Photopea](https://www.photopea.com) 是这份列表中技术上最令人印象深刻的工具，也是最出人意料的。一位开发者——Ivan Kutskir，基本上是独自工作——构建了一款基于浏览器的图像编辑器，可以处理 PSD 文件（Photoshop 的原生格式）、XCF 文件（GIMP 的格式）、Sketch 文件、AI 文件（Adobe Illustrator）、PDF、SVG 以及所有标准网络格式。

打开页面，你立刻进入了编辑器。没有要求输入姓名的加载屏幕，没有套餐选择，没有要跳过的引导教程。它看起来像 2010 年前后的 Photoshop，这没问题，因为 2010 年前后的 Photoshop 功能非常强大。

你可以使用图层、调整图层、智能对象、混合模式、蒙版、滤镜和曲线。不是这些功能的近似实现——而是真正的东西。我曾在没有安装 Photoshop 的机器上，在 Photopea 中打开客户的 PSD 文件，并对其中的文字图层和图层效果进行编辑，一切完好无损。Canva 根本做不到这一点；它是一个模板工具，不是文件编辑器。

商业模式是在免费层级插入广告（还有一个约每月 9 美元的付费无广告方案，为 Kutskir 的个人开发提供资金）。没有任何内容在服务器端处理——你的文件留在浏览器中。Photopea 报告月活跃用户约为 1000 万，对于一个单人项目来说，这是相当惊人的。如果你对一个人如何维护如此规模的东西感到好奇，[Kutskir 的开发博客](https://blog.photopea.com) 值得一读。

请查看 [nologin.tools 上的 Photopea 页面](/tool/photopea-com) 了解完整功能摘要。

**最适合**：编辑分层文件（PSD、XCF、AI）、照片修图、合成，以及任何需要真正图层控制的工作。
**不适合**：如果你需要从模板制作简单的社交媒体图片——那不是它的设计初衷。

## Excalidraw：刻意粗糙的图表工具

[Excalidraw](https://excalidraw.com) 让一切看起来像手绘，这不是偶然的。略显歪斜的形状和手写字体传达出"草稿"的感觉——这对于系统架构图、技术流程图和仍在研究中的线框图来说，正是正确的信号。

无需登录的设计融入了其设计理念。实时协作通过共享 URL 实现，任何参与者无需账号。协作会话是端对端加密的，这意味着即使是 Excalidraw 的服务器也无法看到你在画什么。这对敏感技术工作来说是一个有意义的隐私特性。

Excalidraw 是[基于 MIT 许可证的开源项目](https://github.com/excalidraw/excalidraw)，拥有超过 85,000 个 GitHub 星标——是现有最受欢迎的开源绘图工具之一。它被嵌入 Notion、GitLab、Linear 和数十个其他产品中。该库被广泛使用，这意味着它由专业依赖它的人维护和改进。

与 Canva 的比较很有启发性。如果你在设计评审中带着一张 Canva 精心制作的图表出现，人们会认为你在展示已经完成的东西。带着 Excalidraw 草图出现，对话则保持开放——人们觉得可以自由补充、质疑，把它当作一个起点。在项目的不同阶段，这两种模式都很重要。

[nologin.tools 上的 Excalidraw 完整页面](/tool/excalidraw-com) 有更多详情。

**最适合**：技术图表、系统架构草图、线框图、视觉头脑风暴，当"草稿"就是预期效果时。
**不适合**：如果你需要将被打印或发布的精良成品输出。

## tldraw：真正适合团队的白板工具

[tldraw](https://tldraw.com) 介于 Excalidraw 和 Miro 之间。输出看起来更干净——没有手绘美学——但它仍然是无限画布白板，而不是传统设计工具。形状清晰，界面现代，实时协作方式与 Excalidraw 相同：分享 URL，任何人无需创建账号即可加入。

tldraw 特别擅长的是空间视觉思维。你可以放置便签、绘制好看的箭头、写富文本、嵌入图片，并创建框架来对画布的各个部分进行分组。可以导出为 PNG 和 SVG。整体体验流畅，不像是 2008 年设计的东西。

开发者 SDK 被大量使用——[tldraw 的 GitHub 仓库](https://github.com/tldraw/tldraw) 约有 35,000 个星标，`tldraw` npm 包被嵌入许多需要画布体验的产品中。这意味着底层代码由构建产品的人（而不仅仅是最终用户）审查和改进。

2024 年，tldraw 展示了 AI 辅助画布功能的演示——例如勾勒 UI 线框并让其生成可工作的 HTML。这些功能还处于实验阶段，但方向表明 tldraw 正在投资让画布变得更智能。值得关注。

[nologin.tools 上的 tldraw 完整页面](/tool/tldraw-com) 有更多信息。

**最适合**：团队头脑风暴会议、视觉规划、轻量级线框图，当你需要无需注册摩擦的协作时。
**不适合**：如果你独自工作，需要特定文件格式的输出。

## SVG-Edit：浏览器中的真正矢量编辑

[SVG-Edit](https://svgedit.netlify.app/editor/index.html) 是这里最古老的工具，看起来也是如此。界面实用而陈旧，就像一把实用刀看起来实用而陈旧——它完成工作，不试图在美观方面有所作为。

选择 SVG-Edit 的理由很简单：你需要在不安装软件的情况下创建或编辑 SVG 文件，并且想要干净的输出。SVG-Edit 直接写入 SVG 格式——没有内部表示在导出时被翻译。你绘制的内容就是最终文件中的内容。

这对网络工作很重要。Canva 在某些套餐中可以导出 SVG，但 Canva 的 SVG 输出往往很混乱：大量嵌套的 `<g>` 元素、生成的 ID、冗余属性。相比之下，SVG-Edit 的输出很简洁，这意味着更容易在代码编辑器中手动修改，或通过优化器处理。

它是开源的，已经是多年了——[SVG-Edit GitHub 仓库](https://github.com/SVG-Edit/svgedit) 经历了多次重大重写，仍在维护中。代码库现在是模块化的 ES 模块。不是最迷人的项目，但持续维护且可靠。

[nologin.tools 上的 SVG-Edit](/tool/svgedit-netlify-app-editor-index-html) 有快速摘要。

**最适合**：需要干净 SVG 图标、徽标或插图直接用于代码的网络开发者。
**不适合**：如果你需要友好的界面，或者你在处理照片，或者你对陈旧的 UI 零容忍。

## Haikei：能做到 Canva 做不到的事情的工具

[Haikei](https://haikei.app) 高度专业化，这就是为什么我建议它作为其他工具的补充，而不是替代。它生成装饰性 SVG 背景、blob 形状、波浪分隔线、渐变网格、分层波浪和抽象图案——超过 30 种生成器类型——并导出干净的、可用于生产的 SVG。

工作流程是：选择一个生成器，调整参数，点击随机生成直到效果满意，导出。两分钟搞定。输出是可缩放矢量图形，这意味着在任何屏幕尺寸下都不会像素化，并且文件足够小，如果需要的话可以直接内联到 HTML 中。

Canva 有一些背景选项，但它们是固定尺寸的位图，而不是可配置的矢量资产。如果你在构建一个网站，需要一个与你的品牌颜色相匹配并在各个部分之间平滑过渡的波浪分隔线，Haikei 让你精确控制这一切。这是一种真正不同的能力——不是 Canva 的替代品，而是填补 Canva 留下的空白的工具。

核心生成器无需账号即可免费使用。如果你需要，Pro 层级会增加更多生成器类型。

[nologin.tools 上的 Haikei](/tool/haikei-app) 涵盖了免费层级包含的内容。

**最适合**：为落地页和应用界面创建装饰性 SVG 资产（背景、blob 形状、波浪分隔线）的网络开发者和设计师。
**不适合**：如果你需要通用设计工具，而不是专业的背景生成器。

## 哪一款最适合你

"Canva 替代品"的框架有点误导，因为 Canva 主要是一个附带编辑器的模板库。让 Canva 有用的不是编辑器——而是数千个社交媒体帖子、演示文稿、传单和名片的预建模板。这五款工具中没有一款能复制这一点。

它们拥有的是在特定领域的真正深度：

| 工具 | 主要用途 | 开源 | 导出格式 |
|------|----------|------|----------|
| [Photopea](https://www.photopea.com) | 照片编辑、分层文件 | 否 | PSD, PNG, JPG, SVG, PDF |
| [Excalidraw](https://excalidraw.com) | 技术图表、线框图 | 是 (MIT) | PNG, SVG |
| [tldraw](https://tldraw.com) | 协作白板 | 是 | PNG, SVG |
| SVG-Edit | 干净的 SVG 矢量创作 | 是 | SVG |
| [Haikei](https://haikei.app) | 装饰性 SVG 资产 | 否 | SVG, PNG |

没有一款需要注册。没有一款在服务器上处理你的文件（tldraw 的实时多人同步除外，这正是该功能的意义所在）。所有工具都可以在任何现代浏览器中无需插件运行。

> 注重隐私并不意味着偏执。它只是意味着有意识地选择你留下哪些数据。

结合使用，这些工具涵盖了大多数不特别需要 Canva 模板生态系统的设计工作——并且在不留下账号、设计历史或订阅管理的情况下完成这些工作。[nologin.tools](/tool/nologin-tools) 上更广泛的目录列出了更多这类工具，涵盖多个类别，并且收藏还在持续增长。

基于浏览器、注重隐私、无需注册——这些约束条件证明能够催生出当前正在构建的一些最有趣的软件。那些在不锁定你的情况下生存的工具，往往是真正优秀的工具。

说实话，Canva 的免费版确实不错。这让我有点难以启齿。Canva 精美、快速，数千个模板让设计小白也能做出像样的作品。但它需要注册账号。而这个账号背后，做的事情可比你想象的多。

你一注册，Canva 就拿到了你的邮箱、设计历史、IP 地址、使用习惯，以及一个向你推销每月 15 美元订阅的直接渠道。说白了就是这样——你的注意力和数据，撑起了免费服务。大多数人不假思索地接受了这笔交易。你可能也觉得无所谓。但还是值得了解一下替代品的存在，因为那些完全无需注册的浏览器设计工具，能力已经相当强大了。

Canva 的登录要求还会在你没料到的场景下冒出来：公用电脑、客户的笔记本、临时借用的设备，或者不想让 SaaS 服务商记录你设计内容的敏感项目。还有账号扩散的问题——用了十几个网络工具，每个都要注册，最后就是十几个营销邮件订阅要取消，你的邮箱在十几个地方留了档。

下面这五款无需登录的工具，让你打开标签页、做完设计、导出文件、关掉页面。没有邮件验证，没有新手引导清单，也没有「本月已使用 5 次免费导出中的 3 次」的提示。

## Photopea：比你想象中更接近 Photoshop

[Photopea](https://www.photopea.com) 是这份清单里技术上最令人印象深刻的工具，也是最出乎意料的一个。一位名叫 Ivan Kutskir 的独立开发者，几乎单枪匹马打造了一款浏览器版图像编辑器，支持 PSD（Photoshop 原生格式）、XCF（GIMP 格式）、Sketch、AI（Adobe Illustrator）、PDF、SVG 以及所有常见网络格式。

打开页面就直接进入编辑器。没有让你填名字的加载界面，没有选套餐，没有跳过引导。界面看起来像 2010 年的 Photoshop——没关系，2010 年的 Photoshop 功能已经非常强大了。

图层、调整图层、智能对象、混合模式、蒙版、滤镜、曲线——全都有，而且是真功能，不是近似替代品。我曾在没有安装 Photoshop 的机器上，用 Photopea 打开客户的 PSD 文件，文字图层和图层效果一点没少。Canva 完全做不到这一点，它是模板工具，不是文件编辑器。

商业模式是免费版有广告（付费无广告版约 9 美元/月，支撑 Kutskir 的独立开发）。所有处理都在浏览器内完成，文件不会上传到服务器。Photopea 报告的月活用户约为 1000 万，对于一个人的项目来说相当惊人。[Kutskir 的开发博客](https://blog.photopea.com) 值得一读，了解一个人是怎么维护这么大规模项目的。

查看 [nologin.tools 上的 Photopea 页面](/tool/photopea-com)，了解完整功能介绍。

**适合**：编辑分层文件（PSD、XCF、AI）、照片修图、合成，以及需要真正图层控制的任何工作。
**不适合**：只是想从模板快速做一张社交图——那不是它的用途。

## Excalidraw：故意粗糙的图表工具

[Excalidraw](https://excalidraw.com) 让一切看起来像手绘，这不是偶然。那种略微歪斜的形状和手写字体传递出「草稿」的感觉——这正是系统架构图、技术流程图和还在推敲中的线框图所需要的信号。

无需登录是设计哲学的一部分。实时协作通过共享 URL 实现，所有参与者都不需要账号。协作会话是端对端加密的，就连 Excalidraw 的服务器也看不到你在画什么——这对敏感技术工作来说是真正有意义的隐私保护。

Excalidraw 是[基于 MIT 许可的开源项目](https://github.com/excalidraw/excalidraw)，拥有超过 85,000 个 GitHub Star，是现存最受欢迎的开源绘图工具之一。它被嵌入到 Notion、GitLab、Linear 和数十个其他产品中，意味着项目由真正依赖它的人维护和改进。

和 Canva 的对比很有意思。拿着精心制作的 Canva 图表出现在设计评审中，大家会认为你在展示定稿方案。带着 Excalidraw 草图来，对话就会保持开放——人们更愿意在上面添加、质疑、把它当作起点。不同项目阶段，两种模式都有价值。

[nologin.tools 上的 Excalidraw 完整介绍](/tool/excalidraw-com)。

**适合**：技术图表、系统架构草图、线框图、视觉头脑风暴，当「草稿」是有意为之的信号时。
**不适合**：需要印刷或发布的精美输出。

## tldraw：真正适合团队的白板工具

[tldraw](https://tldraw.com) 介于 Excalidraw 和 Miro 之间。输出更整洁——没有手绘风格——但它仍然是无限画布白板，而不是传统设计工具。形状清晰，界面现代，实时协作方式与 Excalidraw 相同：分享 URL，任何人无需注册即可加入。

tldraw 特别擅长空间视觉思维。你可以放便利贴、画好看的箭头、写富文本、嵌入图片、用框架对画布区域分组。导出 PNG 和 SVG，整体体验流畅，不像 2008 年的产品。

开发者 SDK 使用广泛——[tldraw 的 GitHub 仓库](https://github.com/tldraw/tldraw)约有 35,000 个 Star，`tldraw` npm 包被嵌入到很多需要画布体验的产品里。这意味着底层代码受到真正依赖它做产品的开发者审视和改进，不只是普通用户在用。

2024 年，tldraw 展示了 AI 辅助画布功能的演示——比如画个 UI 线框，然后生成可运行的 HTML。这些功能还在实验阶段，但方向表明 tldraw 在持续投资让画布变得更智能。值得关注。

[nologin.tools 上的 tldraw 完整介绍](/tool/tldraw-com)。

**适合**：团队头脑风暴、视觉规划、协作线框图——无需注册摩擦。
**不适合**：单独工作且需要特定文件格式输出的场景。

## SVG-Edit：真正的浏览器矢量编辑

[SVG-Edit](https://svgedit.netlify.app/editor/index.html) 是这份清单里最老的工具，看起来也确实老。界面实用但过时，就像一把实用但不好看的工具刀——完成工作，没有任何美化。

选 SVG-Edit 的理由很直接：需要在不安装软件的情况下创建或编辑 SVG 文件，并且想要干净的输出。SVG-Edit 直接写入 SVG 格式，没有内部表示需要在导出时转换。你画什么，文件里就是什么。

这对网页工作很重要。Canva 在某些方案下可以导出 SVG，但 Canva 的 SVG 输出往往杂乱：嵌套的 `<g>` 元素、自动生成的 ID、冗余属性。SVG-Edit 的输出相比之下简洁很多，更容易在代码编辑器里手动修改，或者通过优化器处理。

它是开源的，多年来一直如此——[SVG-Edit 的 GitHub 仓库](https://github.com/SVG-Edit/svgedit)经历了多次大改写仍在维护中。代码库现在是模块化的 ES 模块。不是最耀眼的项目，但持续在线且可靠。

[nologin.tools 上的 SVG-Edit 简介](/tool/svgedit-netlify-app-editor-index-html)。

**适合**：需要干净 SVG 图标、Logo 或插图直接用于代码的网页开发者。
**不适合**：需要友好界面，或处理照片，或不能忍受过时 UI 的用户。

## Haikei：能做到 Canva 做不到的事

[Haikei](https://haikei.app) 高度专业化，所以我建议把它和其他工具搭配使用，而不是替代。它能生成装饰性 SVG 背景、Blob 形状、波浪分割线、渐变网格、叠层波浪和抽象图案——30 多种生成器类型——导出干净的可直接投入生产的 SVG。

工作流程是：选择生成器，调整参数，点击随机直到满意，导出。两分钟搞定。输出是可缩放矢量图形，任何屏幕尺寸下无像素化，文件足够小，可以直接内联到 HTML 里。

Canva 有一些背景选项，但它们是固定尺寸的位图，不是可配置的矢量资源。如果你在做网站，需要一个与品牌色匹配、在区块间平滑过渡的波浪分割线，Haikei 让你精确控制这一切。这是真正不同的能力——不是 Canva 的替代品，而是填补 Canva 留下空白的工具。

核心生成器无需任何账号免费使用。需要更多生成器类型的话，Pro 版提供升级。

[nologin.tools 上的 Haikei 介绍](/tool/haikei-app)，了解免费版包含的内容。

**适合**：创建装饰性 SVG 资源的网页开发者和设计师——背景、Blob 形状、落地页和应用界面的波浪分割线。
**不适合**：需要通用设计工具而非专门背景生成器的用户。

## 选哪个？

「Canva 替代品」这个框架其实有点误导，因为 Canva 本质上是一个附带编辑器的模板库。让 Canva 有用的不是编辑器本身，而是那些针对社交媒体、演示文稿、传单和名片的数千个预制模板。这五款工具都没有复制那些。

它们有的是各自领域的真正深度：

| 工具 | 主要用途 | 开源 | 导出格式 |
|------|----------|------|----------|
| [Photopea](https://www.photopea.com) | 照片编辑、分层文件 | 否 | PSD、PNG、JPG、SVG、PDF |
| [Excalidraw](https://excalidraw.com) | 技术图表、线框图 | 是（MIT） | PNG、SVG |
| [tldraw](https://tldraw.com) | 协作白板 | 是 | PNG、SVG |
| SVG-Edit | 干净的 SVG 矢量创作 | 是 | SVG |
| [Haikei](https://haikei.app) | 装饰性 SVG 资源 | 否 | SVG、PNG |

全部无需注册。文件处理均在本地进行（tldraw 的实时多人同步除外，那本来就是它的核心功能）。所有工具在现代浏览器中无需插件即可运行。

> 隐私友好并不意味着偏执。只是有意识地选择你留下了哪些数据。

合理搭配使用，这些工具能覆盖大多数不特别依赖 Canva 模板库的设计工作——而且不留下账号、设计历史或订阅负担。[nologin.tools](/tool/nologin-tools) 上还有更多同类工具，横跨各个分类，且持续增加。

纯浏览器、隐私友好、无需注册——这些约束条件，反而催生了当下最有趣的一批软件。那些不靠锁定用户生存的工具，往往真的很好用。

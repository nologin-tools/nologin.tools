Die meisten Browser-Tools finden ihr Publikum auf dieselbe Weise: ein viraler Tweet, ein Reddit-Frontpage-Treffer, und plötzlich bookmarken alle dieselbe Handvoll Apps. Tools, die dieses Zeitfenster verpassen, bleiben unsichtbar — selbst wenn sie exzellent sind. Selbst wenn sie seit Jahren still und leise laufen.

Das ist ein erster Blick auf sechs kostenlose Browser-Tools, die ganz ohne Konto funktionieren. Keine E-Mail-Adresse, keine Kontoanmeldung, kein Passwort zum Vergessen. Manche sind neu; manche existieren seit Jahren ohne ein breites Publikum gefunden zu haben. Alle tun etwas wirklich Nützliches, und keines fragt nach einer Registrierung, bevor es dich loslegen lässt.

## Was „kein Login erforderlich" hier wirklich bedeutet

Es gibt ein Spektrum. Manche Tools bewerben sich als „ohne Login", sperren aber nützliche Features hinter einer Registrierung oder lassen deine Arbeit nach 24 Stunden ablaufen, um dich unter Druck zu setzen, ein Konto zu erstellen. Darum geht es in dieser Liste nicht.

Jedes Tool hier ist ohne Registrierung voll funktionsfähig — was du beim ersten Laden siehst, das bekommst du. Mehrere laufen komplett in deinem Browser, was bedeutet, dass deine Daten nie einen Server erreichen. Das ist ein Datenschutzgewinn, aber es bedeutet auch, dass diese Tools offline funktionieren, sofort laden und nicht kaputt gehen, wenn ein Unternehmen pivotiert oder schließt.

Wenn du eine breitere Liste verifizierter No-Login-Tools möchtest, kuratiert [nologin.tools](/tool/nologin-tools) Hunderte davon nach Kategorien. Aber die sechs unten verdienen es, besonders erwähnt zu werden.

## led.run — Jeden Bildschirm in eine scrollende Anzeige verwandeln

Wenn du scrollenden Text auf einem Bildschirm anzeigen musst — bei einem Konzert, während eines Workshops, auf einem Monitor an einem Empfangstisch — erfordern die meisten Optionen entweder dedizierte Hardware oder eine Registrierungs-App. [led.run](/tool/led-run) ist ein kostenloses Browser-Tool, das beides überspringt.

Öffne die Seite, tippe deinen Text, und dein Browserfenster wird zu einem scrollenden LED-Stil-Ticker im Vollbildmodus. Du kannst Schriftgröße, Scrollgeschwindigkeit, Textfarbe und Richtung anpassen. Das Rendering ist scharf und kontrastreich, von der anderen Seite eines Raums lesbar.

Das klingt nischenförmig. Aber wenn du schon mal versucht hast, in letzter Minute vor einer Veranstaltung schnell etwas zusammenzustückeln, ist das genau die Art von Tool, das du jemandem als Link schickst und das sie sofort bookmarken. Keine Installation, keine Registrierung, kein Download.

## tmp.tf — Das einfachste geräteübergreifende Clipboard

Das Szenario: Du bist auf deinem Arbeits-Laptop und musst ein Text-Snippet — eine URL, einen Code-Block, eine Telefonnummer — auf dein Handy oder einen anderen Computer übertragen. Du hast keine gemeinsame Clipboard-App konfiguriert, und dir eine E-Mail zu schicken fühlt sich absurd an.

[tmp.tf](/tool/tmp-tf) ist ein browser-basiertes temporäres Clipboard. Text einfügen, kurze URL bekommen, diese URL auf dem anderen Gerät öffnen. Der Inhalt ist sofort verfügbar. Stell dir das als Pastebin vor, das für den Einmalgebrauch gedacht ist: standardmäßig flüchtig, kein gespeicherter Verlauf, kein Konto nötig.

Was es von generischen Pastebins unterscheidet, ist die Designabsicht. Das ist kein Tool zum Veröffentlichen von Code-Snippets oder öffentlichen Inhaltssharing — es ist für schnelle persönliche Synchronisation gebaut.

## Datasette Lite — SQL-Abfragen gegen eigene Dateien, im Browser

Wenn du schon mal eine CSV- oder SQLite-Datei hattest und Abfragen dagegen ausführen wolltest, ohne einen Datenbankserver einrichten zu müssen, ist Datasette Lite es wert, bekannt zu werden. Es führt [Datasette](https://datasette.io/) — Simon Willisons leistungsstarkes Datenexplorationstool — komplett in deinem Browser mit [WebAssembly](https://developer.mozilla.org/en-US/docs/WebAssembly) aus. Keine Installation, kein Backend, kein Konto.

Du lädst eine SQLite-Datenbankdatei von deinem lokalen Rechner, und die vollständige Datasette-Oberfläche erscheint: Tabellen-Browsing, facettiertes Filtering, SQL-Abfrageausführung und Datenexport. Deine Daten verlassen deinen Browser nie. Die SQLite-Engine kompiliert direkt zu WebAssembly, was bedeutet, dass eine produktionstaugliche Datenbank komplett auf deinem Gerät läuft.

Für einen Datenjournalisten, Analysten oder Entwickler, der eine schnelle Exploration eines lokalen Datensatzes macht, ist das erheblich. Die meisten vergleichbaren Tools erfordern entweder ein Server-Setup oder Cloud-Speicher und Kontoerstellung. [Datasette Lite](/tool/lite-datasette-io) braucht keines von beidem.

> „Die besten Tools sind die, bei denen man vergisst, dass man Infrastruktur verwendet."

Dieses Zitat trifft hier zu. Was sonst ein mehrstufiges Setup wäre — SQLite installieren, eine Datenbank laden, Datasette konfigurieren, einen Server öffnen — wird zu einem Browser-Tab.

## Markmap — Deine Markdown-Gliederung wird zur Mindmap

Strukturierte Notizen zu schreiben ist nützlich. Sie als verbundenes Diagramm zu sehen ist etwas anderes. [Markmap](/tool/markmap-js-org) konvertiert eine Markdown-Gliederung — Überschriften, Unterüberschriften, verschachtelte Aufzählungen — in eine interaktive, zoombare Mindmap. In Echtzeit. Kein Drag-and-Drop, keine Knotenplatzierung, keine Registrierung.

Der Workflow: Markdown im linken Panel schreiben, und die Mindmap baut sich rechts von selbst auf. Du kannst Äste ein- und ausklappen, zoomen und schwenken, und das Ergebnis als SVG exportieren. Besonders gut geeignet, um eine Dokumentstruktur in etwas zu verwandeln, das du in einer Präsentation zeigen oder als visuellen Überblick teilen kannst.

Im Gegensatz zu Mindmapping-Tools wie MindMeister oder Miro — die dich zur Registrierung drängen, bevor du überhaupt angefangen hast — ist Markmap Open Source und ab dem ersten Besuch voll funktionsfähig.

Markmap ist auch eine gute Antwort auf die Frage, warum No-Login-Tools über Bequemlichkeit hinaus wichtig sind. Wenn ein Tool komplett im Browser läuft und seinen Zustand als Text verwaltet, gibt es kein Konto zu verlieren, keinen Export erforderlich vor Ablauf eines Abonnements, kein Lock-in. Der Inhalt gehört dir von Anfang an.

## Goblin.tools — KI-Aufgabenzerlegung ohne Konto

Standard-To-do-Apps setzen voraus, dass du bereits weißt, wie du eine Aufgabe in Schritte unterteilst. Für viele Menschen — besonders solche mit ADHS oder anderen Herausforderungen bei exekutiven Funktionen — ist genau diese Annahme das Problem. [Goblin.tools](/tool/goblin-tools) basiert auf einem anderen Modell: Du beschreibst, was passieren muss, und die KI zerlegt es.

Die Kernfunktion, „Magic To-Do" genannt, nimmt eine vage Aufgabenbeschreibung („die Präsentation vorbereiten", „diese E-Mails beantworten") und generiert eine konkrete Schritt-für-Schritt-Liste. Du kannst die Granularität anpassen — mehr Schritte für überwältigende Aufgaben, weniger für solche, die nur einen Ausgangspunkt brauchen. Es gibt auch ein separates Tool zur Schätzung der Aufgabendauer, eines zum Konvertieren eines Textblocks in eine Checkliste und eines zur Analyse des Tons einer geschriebenen Kommunikation.

Bemerkenswert ist, dass die Kerntools ohne Registrierung funktionieren. KI-gestützte Features kommen normalerweise hinter einer Konto-Mauer — für Rate-Limiting und Datenerfassung. Goblin.tools ist eine Ausnahme.

Das ist ein schmaler Fokus, gut umgesetzt. Das ist das Muster bei den meisten Tools in dieser Liste.

## SiteAge — Wann eine Website geboren wurde

Eine unbekannte Website zu evaluieren — eine Nachrichtenquelle, einen Anbieter, ein Tool, dem du gleich eine Aufgabe anvertraust — bedeutet oft zu wissen, wie alt sie ist. Wie lange gibt es sie schon? Wann wurde sie zuerst indexiert? Hat sie sich stark verändert?

[SiteAge](/tool/siteage-org) fragt die [Wayback Machine des Internet Archive](https://web.archive.org/) ab und gibt das ungefähre Gründungsdatum einer Site sowie einen Zeitstrahl zurück, wie sich ihr Aussehen über die Jahre verändert hat. URL eingeben, Geschichte bekommen. Keine Registrierung, kein API-Key.

Das ist ein schmales Tool, das eine spezifische Sache tut, aber eine echte Lücke füllt. Die Wayback Machine hat die Daten; SiteAge macht sie zugänglich, ohne dass du manuell historische Snapshots durchsuchen musst. Für Forscher, Journalisten oder jeden, der Due Diligence an einer unbekannten Domain macht, ist das eine Ein-Schritt-Abkürzung.

Die Unterscheidung „Daten bleiben lokal" zählt mehr, als es scheint. Tools wie Datasette Lite und Markmap verwalten deinen Inhalt komplett auf deinem Gerät — es gibt keinen Server, der deine Datenbankdateien oder Notizen empfängt. Das ist ein bedeutsamer Datenschutzunterschied gegenüber Tools, die Daten in der Cloud verarbeiten, selbst wenn beide behaupten „kein Login erforderlich".

## Warum No-Login-Tools immer besser werden

Es gibt einen strukturellen Grund für die Existenz dieser Tools: WebAssembly hat es praktikabel gemacht, ernsthaftes Software — Compiler, Datenbanken, Bildverarbeiter — direkt im Browser zu betreiben. Was früher einen Server erforderte, läuft jetzt lokal. Das verändert die Ökonomie des Tool-Bauens, denn ein rein clientseitiges Tool hat keine laufenden Infrastrukturkosten, was bedeutet: kein Druck zur Monetarisierung über Konten und Daten.

Die Open-Source-Community hat das voll ausgenutzt. Tools wie Datasette Lite und Markmap werden von Entwicklern gepflegt, die sie veröffentlichen, weil die Tools nützlich sind, nicht weil sie Teil eines SaaS-Funnels sind. Diese Interessenausrichtung produziert Software, die den Nutzer wirklich respektiert.

Nicht jedes Tool in dieser Liste ist Open Source. Aber das No-Login-Prinzip — erst Zugang, nichts fragen — gilt für alle. Das sind Tools, die darauf vertrauen, dass du sie nutzt, ohne wissen zu müssen, wer du bist.

Wenn du feststellst, dass du eines davon regelmäßig verwendest, lohnt es sich, im [vollständigen Verzeichnis](/tool/nologin-tools) nach verwandten Tools in derselben Kategorie zu suchen. Es gibt noch mehr davon.
